### 基本步骤

-- 从bitbucket中取项目。 `git clone git@bitbucket.org:umftech/restdemo.git`
-- 通过 mvn命令， 把项目转为eclipse WTP项目. 具体命令： `mvn eclipse:eclipse -Dwtpversion=2.0`
